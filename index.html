The Classic Chaser-Runner Game

The Classic Chaser-Runner Game

  1. Start with a graph, a chaser player, and a runner player.
  2. The chaser player places chaser(s) on vertices of the graph.
  3. The runner player places the runner on a vertex of the graph.

The Classic Chaser-Runner Game

  1. Start with a graph, a chaser player, and a runner player.
  2. The chaser player places chaser(s) on vertices of the graph.
  3. The runner player places the runner on a vertex of the graph.
  1. Alternating taking turns, starting with the chaser player.
  2. On a players turn, for each piece they control, they may choose to keep them where they are, or move them to an adjacent vertex.
  3. If a chaser moves onto a runner, or vice versa, the chaser wins.
  4. The runner wins by having a strategy to indefinitely avoid the chasers.

The Classic Chaser-Runner Game

Given a graph \(G\), the chaser number of \(G\) denoted \(c(G)\) denotes the minimum number of chasers needed to ensure a winning strategy for the chaser player.

  • \(c(K_2)=1\).
  • \(c(P_5)=1\).
  • \(c(T)=1\) for all trees \(T\).
  • \(c(C_n)=2\) for all cycles \(C_n, n\geq 3\).
  • \(c(K_n)=1\) for all complete graphs \(K_n, n\geq 3\).
  • \(c(K(5,2))=3\) (also called the Petersen Graph).

The Sneaky Active Chaser-Runner Game

  2. On a players turn, for each piece they control, they must move them to an adjacent vertex.
  3. If a chaser moves onto a runner the chaser wins. If a runner moves onto the chaser, they are safe.

The Sneaky Active Chaser-Runner Game

Given a graph \(G\), the sneaky active chaser number of \(G\) denoted \(c_{SA}(G)\) denotes the minimum number of chasers needed to ensure a winning strategy for the chaser player.

  • \(c_{SA}(K_2)=2\).
  • \(c_{SA}(T)=2\) for all trees \(T\).
  • \(c_{SA}(C_4)=2\).
  • \(c_{SA}(C_{2k+1})=2\) for all \(k\geq 1\).
  • \(c_{SA}(C_{2k})=4\) for all \(k\geq 3\).
  • \(c_{SA}(K_n)=2\) for all \(n\geq 3\).

Theorem (C, Scull) sneaky-active chaser number is a \(\times-\)homotopy invariant.

The Graph Product \(G\times H\) is the cartesian products of the vertices of \(G, H\) where \((v_1, w_1)\sim (v_2, w_2)\) whenever \(v_1\sim w_2\) and \(v_2\sim w_1\).

Theorem (C,Scull) \[c_{SA}(X\times Y)= \ldots \]

  • \( c_{SA}(X)+c_{SA}(Y)-1\) if \(X, Y\) are not bipartite.
  • \( 2c_{SA}(Y)+c_{SA}(Y)-2\) if \(X\) is not bipartite, but \(Y\) is.
  • \( 2c_{SA}(X)+2c_{SA}(Y)-4\) if \(X, Y\) are both bipartite.
